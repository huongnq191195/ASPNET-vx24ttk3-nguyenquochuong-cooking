## Báo cáo tiến độ Tuần 2

* **Xây dựng cấu trúc dự án**: Tổ chức lại cây thư mục theo yêu cầu (đưa code vào thư mục `src`).

* **Phát triển tính năng**:
    * Hoàn thiện giao diện người dùng (Đăng ký, Đăng nhập, Tìm kiếm).
    * Xây dựng chức năng quản lý công thức: Đăng bài, Sửa bài, Duyệt bài (chỉ Admin); Hiển thị chi tiết món ăn.
    * Triển khai tính năng: Like, Comment, Thêm món ăn vào kho yêu thích.
    * Tính năng phụ: Admin có thể thay đổi hình nền, thêm danh mục mới.

* **Xây dựng cơ sở dữ liệu**: Thiết lập model dữ liệu, cấu hình Entity Framework để tự động tạo lược đồ (schema) database.

* **Công việc tồn đọng**: Cần hoàn thiện nhập liệu dữ liệu mẫu (Seeding data) để phục vụ báo cáo.
