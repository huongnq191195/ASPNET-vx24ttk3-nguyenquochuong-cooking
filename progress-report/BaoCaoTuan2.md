## Báo cáo tiến độ Tuần 2

* **Xây dựng cấu trúc dự án**:
    * Tổ chức lại cấu trúc thư mục của dự án theo yêu cầu học phần.
    * Thiết lập môi trường phát triển bằng ASP.NET Core MVC và SQL Server LocalDB.

* **Phát triển chức năng hệ thống**:
    * Hoàn thiện chức năng Đăng ký và Đăng nhập tài khoản.
    * Xây dựng chức năng đăng công thức nấu ăn.
    * Xây dựng chức năng chỉnh sửa và xóa công thức.
    * Xây dựng chức năng duyệt bài đăng dành cho quản trị viên (Admin).
    * Xây dựng chức năng xem chi tiết công thức nấu ăn.
    * Hoàn thiện chức năng tìm kiếm công thức theo từ khóa.

* **Triển khai các tính năng tương tác**:
    * Xây dựng chức năng thích (Like) công thức.
    * Xây dựng chức năng bình luận (Comment) công thức.
    * Xây dựng chức năng lưu công thức yêu thích (Saved Recipes).

* **Xây dựng cơ sở dữ liệu**:
    * Thiết kế các bảng dữ liệu: Users, Recipes, Categories, Comments, UserLikes và SavedRecipes.
    * Cấu hình Entity Framework Core kết nối với SQL Server LocalDB.
    * Tạo Migration và khởi tạo cơ sở dữ liệu.
    * Kiểm tra hoạt động lưu trữ và truy xuất dữ liệu.

* **Xây dựng dữ liệu mẫu**:
    * Tạo tài khoản Admin và các tài khoản người dùng mẫu.
    * Thêm các công thức nấu ăn mẫu để kiểm thử hệ thống.
    * Bổ sung dữ liệu bình luận, lượt thích và danh sách yêu thích.

* **Công việc còn lại**:
    * Bổ sung thêm dữ liệu mẫu và hình ảnh minh họa cho các món ăn.
    * Kiểm thử toàn bộ chức năng của hệ thống.
    * Hoàn thiện báo cáo đồ án và hướng dẫn cài đặt hệ thống.
    * Chuẩn bị dữ liệu và source code để nộp trên GitHub.
