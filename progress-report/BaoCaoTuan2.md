\## Báo cáo tiến độ Tuần 2

&#x20;   \* \*\*Xây dựng cấu trúc dự án\*\*: Tổ chức lại cây thư mục theo yêu cầu (đưa code vào thư mục `src`).

&#x20;   \* \*\*Phát triển tính năng\*\*:

&#x20;       \* Hoàn thiện giao diện người dùng (Đăng ký, Đăng nhập, Tìm kiếm).

&#x20;       \* Xây dựng chức năng quản lý công thức: Đăng bài, hiển thị chi tiết món ăn.

&#x20;       \* Triển khai tính năng Bình luận (Comment).

&#x20;   \* \*\*Xây dựng cơ sở dữ liệu\*\*: Thiết lập model dữ liệu, cấu hình Entity Framework để tự động tạo lược đồ (schema) database.

&#x20;   \* \*\*Công việc tồn đọng\*\*: Cần hoàn thiện nhập liệu dữ liệu mẫu (Seeding data) để phục vụ báo cáo.
