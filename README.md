# ĐỒ ÁN MÔN HỌC: CHUYÊN ĐỀ ASP.NET

### 1. Giới thiệu đề tài
Website giúp người dùng có thể tìm kiếm, xem và chia sẻ các công thức nấu ăn ngon hàng ngày.

### 2. Các chức năng chính (Dự kiến)
* Đăng ký, đăng nhập tài khoản.
* Xem danh sách món ăn, tìm kiếm món ăn theo nguyên liệu.
* Thành viên đăng bài chia sẻ công thức nấu ăn mới.
* Bình luận dưới bài viết.
